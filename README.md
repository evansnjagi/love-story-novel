# love-story-novel

A professional novel project written step by step in GitHub Codespaces.

 The story follows two young teenagers who fall in love. Just when the man believes he has found the love of his life, he loses her when everything fails to go as planned.


## Project structure

This project uses a modular structure with different directories. Each folder contains files related to a specific part of the novel.


Example of the tree structure: 

```bash
├── README.md
├── chapters
│   └── chapter01.md
├── characters
│   └── main_characters.md
├── notes
│   └── ideas.md
├── outline.md
└── world
    └── settings.md

5 directories, 6 files
```

Folder details:

- chapters/ -> Story chapters.
- characters/ -> Character profiles.
- world/ -> Settings, locations and timelines.
- notes/ -> Build ideas and brainstorming.
- outline.md -> Story roadmap.
